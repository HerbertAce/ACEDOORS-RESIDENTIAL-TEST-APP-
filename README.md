# AceDoors Residential v0.2.2

GitHub Pages test build for Residential quoting.

## Added

- New Build / Retrofit installation selection.
- Fixing substrate selection: Timber, Steel, or Concrete / Block.
- Automatic steel and masonry fixing surcharges.
- Automatic Retrofit Fee.
- Existing door removal questions for material and door type.
- Automatic steel or timber removal charge.
- Automatic dumping / disposal charge.
- Removal planning notes for technician numbers and lifting equipment.
- Force Battery Backup, Safety Beams, Wi-Fi Control, and Auto-Lock placeholders.
- Price-TBC accessories can be included for site confirmation without blocking quote generation.

## Test status

This is not approved for live quoting. Verify all pricing and rules before customer use.

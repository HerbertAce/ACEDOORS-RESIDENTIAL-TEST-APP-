# AceDoors Residential v0.2.3

Door Builder recovery release.

## Fixed
- Door configuration no longer disappears after upgrading from an earlier saved test build.
- Existing browser-saved door records are migrated to the current format.
- Legacy motor names and missing door fields are repaired automatically.
- A damaged individual door record no longer prevents the full quote from loading.
- Pricing falls back safely to Retail if an old saved tier is unavailable.

## Deployment
Replace all files in the GitHub Pages repository with the contents of this folder.

# AceDoors Residential v0.2.7

Adds a GitHub Pages-compatible digital acceptance test workflow.

- Office can copy or preview a customer acceptance link after marking a quote sent.
- Customer can review the quote, open Terms & Conditions, sign and download an acceptance receipt.
- Office can import the receipt into the job to unlock the Pro Forma / PO stage.

This is a static test workflow. Automatic cross-device syncing and email delivery require the future shared backend.

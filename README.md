# AceDoors Residential v0.3.0

Flat GitHub Pages test release.

Key fixes:
- Blank doors no longer create installation pricing.
- Colour selection moved below the configuration fields.
- Internal Roller calculation notes removed from the quoting UI.
- Lifting equipment remains visible on customer documents when included.
- Digital acceptance no longer downloads a receipt file.
- Same-browser acceptance syncs automatically and generates the non-account pro forma.

Cross-device customer acceptance still requires a shared backend/database.

# AceDoors Residential v0.2.9

GitHub Pages flat release.

## Main changes
- Clean quote start: no prefilled Garage 1 or live pricing.
- Completed garages collapse when another door is added.
- Roller motor pricing from the April 2024 Master RD Parts & Accessories list.
- Added Ace RD1000, Merlin SilentDrive Essential, Merlin SilentDrive Elite and Grifco LR Drive.
- Manual Roller Doors automatically include a priced D-Handle.
- Centre Lock, Slide Lock, Direct Drive Hand Chain, Planetary Hand Chain, Reverse Roll and Bird Brush are selectable Roller options.
- Genie Material Lifter and Scissor Lift appear as included items on the quote.
- Roller labour hours and rates are hidden from the customer workflow and maintained under Admin / Cost Settings.
- Browser title, header and build stamp all use v0.2.9.

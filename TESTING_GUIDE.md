# v0.2.2 Testing Guide

Please test these scenarios:

1. New Build + Timber fixing: no fixing surcharge.
2. New Build + Steel fixing: steel fixing cost auto-added.
3. New Build + Concrete / Block: masonry fixing cost auto-added.
4. Retrofit + no removal: Retrofit Fee only.
5. Retrofit + steel sectional removal + disposal: Retrofit Fee, steel removal, and dumping auto-added.
6. Retrofit + timber tilt removal: timber removal selected and second-tech planning note shown.
7. Retrofit + steel roller removal: lifting equipment, scissor lift, and second-tech note shown.
8. Force motor: only Force accessories display.
9. Select a Price TBC Force accessory and confirm the quote still generates with a TBC note.

Check that all automatically selected extras appear in pricing and on the customer quotation.
